# ShoppingSite - E-Commerce Website

A minimal, fast, and responsive e-commerce website built with pure HTML, CSS, and JavaScript.

## Features

- **Dark Mode**: Sleek modern dark theme with solid colors.
- **Product Store**: Browsing via FakeStoreAPI.
- **Product Details**: Zoom view, size selection, quantity management.
- **Cart System**: Fully functional cart with LocalStorage persistence.
- **Checkout**: Order summary and mock checkout process.

## Tech Stack

- **HTML5** - Semantic structure
- **CSS3** - Modern dark theme (#0f172a, #8b5cf6), Responsive design
- **JavaScript** - Pure Vanilla JS (ES6+), Async/Await
- **API** - FakeStoreAPI

## How to Run

1. Open the folder in VS Code.
2. Run `npx serve .` 
3. Open `http://localhost:3000`
